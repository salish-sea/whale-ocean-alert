# whale-ocean-alert
Content and code to support the Whale Alert and Ocean Alert mobile apps in the Salish Sea region and range of the SRKWs.

1. [Google sheet with HTML+images for zone notifications via a pop-up](https://docs.google.com/spreadsheets/d/18SWEHxLyWd7xaXwC9UQFBykCssJp8cHFcqmDhqmZDc8/edit#gid=0)
2. Tools for creating/editing polygons -- [geojson.io](https://geojson.io/#map=6.44/49.18/-124.952)
3. Alert zones for the Salish Sea: [v1](whale-alert-areas-v1.geojson)
